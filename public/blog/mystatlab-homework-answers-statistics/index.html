Worked Examples

Example 1

Problem: Calculate the mean of the following dataset: 4, 8, 6, 5, 3.
Step 1: List the data points: 4, 8, 6, 5, 3.
Step 2: Count the number of data points, n = 5.
Step 3: Sum the data points: 4 + 8 + 6 + 5 + 3 = 26.
Step 4: Apply the formula for mean: Mean = \frac{26}{5} = 5.2.

Example 2

Problem: A dataset contains the values: 12, 15, 14, 10, 18, 20. Calculate the variance.
Step 1: Calculate the mean: Mean = \frac{12 + 15 + 14 + 10 + 18 + 20}{6} = 14.83.
Step 2: Subtract the mean from each data point and square the result: (12-14.83)^2 = 8.00, (15-14.83)^2 = 0.03, (14-14.83)^2 = 0.69, (10-14.83)^2 = 23.29, (18-14.83)^2 = 10.09, (20-14.83)^2 = 26.69.
Step 3: Sum the squared differences: 8.00 + 0.03 + 0.69 + 23.29 + 10.09 + 26.69 = 68.79.
Step 4: Divide by the number of data points minus one: Variance = \frac{68.79}{6-1} = 13.76.
