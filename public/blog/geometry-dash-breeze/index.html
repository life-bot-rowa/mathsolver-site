Worked Examples

Example 1

Problem: Let's solve a geometry problem to apply what we've learned. We have a triangle with vertices at A(0,0), B(4,0), and C(0,3).
Step 1: To find its area, we use the formula for the area of a triangle with vertices at (x1, y1), (x2, y2), and (x3, y3): area = 1/2 * abs(x1(y2-y3) + x2(y3-y1) + x3(y1-y2)).
Step 2: Substituting in the values, we get area = 1/2 * abs(0(0-3) + 4(3-0) + 0(0-0)) = 1/2 * abs(0 + 12 + 0) = 6 square units.
Step 3: To find the perimeter, we need to calculate the lengths of each side using the distance formula.
Step 4: The distance between A and B is sqrt((4-0)^2 + (0-0)^2) = sqrt(16) = 4.
Step 5: The distance between B and C is sqrt((4-0)^2 + (0-3)^2) = sqrt(25) = 5.
Step 6: The distance between C and A is sqrt((0-0)^2 + (3-0)^2) = sqrt(9) = 3.
Step 7: Adding these gives the perimeter: 4 + 5 + 3 = 12 units.

Problem: Now, let's find the distance between points P(2, 3) and Q(6, 7) using the distance formula.
Step 1: The formula is distance = sqrt((x2 - x1)^2 + (y2 - y1)^2).
Step 2: Substituting the given points, we have distance = sqrt((6 - 2)^2 + (7 - 3)^2) = sqrt(4^2 + 4^2) = sqrt(16 + 16) = sqrt(32) = 4sqrt(2).
